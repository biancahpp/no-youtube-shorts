Privacy Policy – No YouTube Shorts

Last updated: March 2026

No YouTube Shorts (“the extension”) is a browser extension that hides YouTube Shorts from the feed. This policy explains how the extension handles information.

Data collection

We do not collect any data. The extension does not send information to any server. It does not use analytics, tracking, or telemetry.

What the extension does

The extension runs only on YouTube pages. It uses a content script to find and remove or hide Shorts-related elements in the page’s DOM (e.g. Shorts sections and the Shorts shelf). All of this happens locally in your browser. No data is stored, read from your account, or transmitted elsewhere.

Permissions

The extension requests access only to run on *://*.youtube.com/*. It does not request access to your browsing history, cookies, or other sites.

Changes

If this policy changes, we will update the “Last updated” date above. Continued use of the extension after changes means you accept the updated policy.

Contact

For questions about this privacy policy, please open an issue on the extension’s GitHub repository.
